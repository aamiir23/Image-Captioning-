## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/Apaark13/GraphScape.git
   ```

2. Install dependencies:

   ```bash
   cd GraphScape
   npm install
   ```

3. Run the application:

   ```bash
   npm start
   ```
